Where do I get these files?
  1. Spotify → Account → Privacy settings.
  2. Request Extended streaming history (arrives by email, can take a few days).
  3. Unzip it and select every Streaming_History_Audio_*.json file here.

Genres and cover art aren't in the export — they only appear if you run the optional local enrich.py step (see the project README).
